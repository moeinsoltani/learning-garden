# Phase 17: Advanced Routing & Data-Center Fabrics

Routing at scale: how BGP is run inside large networks and data centers. Route reflectors tame
iBGP, EVPN replaces flood-and-learn overlays with a control plane, segment routing moves path
decisions to the source, and anycast serves one address from many places. Builds on Phase 6
(FRR/BGP/OSPF) and Phase 4 (VXLAN).
