# Phase 9: Traffic Control — tc, qdisc, Filters

Traffic control is how the kernel queues, schedules, shapes, and prioritizes packets on egress.
This phase builds from the qdisc model up through rate limiting, network emulation, hierarchical
shaping with HTB, classification with filters, and measuring it all with iperf3.
