# Phase 6: Dynamic Routing — FRR

Static routes don't adapt when a link fails. Dynamic routing protocols let routers discover
topology and reroute automatically. This phase introduces FRR (Free Range Routing) and builds
self-healing OSPF networks and multi-AS BGP peering inside namespaces.
