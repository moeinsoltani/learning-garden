# Phase 11: eBPF & BPF Maps

eBPF lets you run verified programs inside the kernel's datapath — the technology behind Cilium,
modern observability, and high-performance packet processing. This phase covers eBPF fundamentals,
BPF maps for sharing state, and XDP for the earliest, fastest packet hook.
