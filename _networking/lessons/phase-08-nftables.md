# Phase 8: Firewalling — nftables

nftables is the modern Linux firewall, replacing iptables. This phase covers its architecture
(tables, chains, hooks), how to build a real stateful firewall, how to write efficient rules
with sets and maps, and how to offload established flows with a flowtable.
