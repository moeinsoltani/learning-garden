# Phase 1: Sentence Mechanics

This phase builds the foundation: writing accurate sentences quickly. It targets
the highest-frequency error patterns for non-native English speakers — the
sentence core, articles (*a/an/the*/nothing), verb tenses, subject–verb
agreement, prepositions, and connecting ideas without run-ons — because fixing
these three things especially (agreement, articles, and words that drop out of
longer sentences) makes everything you write noticeably more accurate.

Every lesson uses **rewrite drills**: you get realistic, broken software-workplace
messages, you rewrite them, then reveal the model rewrite with the reasoning. Each
lesson ends with a **phrase bank** you can start using the same day.
