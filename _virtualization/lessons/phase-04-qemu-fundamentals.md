# Phase 4: QEMU Fundamentals

Time to drive. This phase makes you fluent in QEMU itself: every flag of a boot
command line, the virtual motherboard (machine types) and accelerators, firmware
and the boot path (SeaBIOS vs OVMF/UEFI), the HMP/QMP monitors that introspect and
control a live VM, and getting console/display output. Everything here is what
libvirt (Phase 10) later generates for you — but you should understand the raw
tool first.
