# Phase 13: Tracing & Debugging

The course closes with the skills that make everything else useful: finding
out what a running system is actually doing. perf answers "where is the CPU
time going?", ftrace/kprobes trace any kernel function as it runs, core dumps
enable post-mortems on dead processes — and the capstone assembles the whole
toolbox into a debugging methodology, a structured hunt through misbehaving
systems that ties together every subsystem you've studied.
