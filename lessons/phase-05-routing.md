# Phase 5: Routing

Layer 2 keeps you on one segment. Layer 3 — routing — is how packets cross *between* networks.
This phase covers how the kernel chooses a route, how to turn a namespace into a router, and
how to route by source/mark (policy routing) instead of just destination.
