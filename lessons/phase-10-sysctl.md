# Phase 10: Kernel Network Parameters — sysctl

sysctl is how you tune the kernel's networking behavior at runtime — forwarding, reverse-path
filtering, buffer sizes, congestion control, and more. This phase covers the most important
tunables and how to make changes persist across reboots and behave correctly per namespace.
