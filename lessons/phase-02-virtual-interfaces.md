# Phase 2: Virtual Interfaces

Real networks are built from cables and NICs. Linux builds the same shapes in software.
This phase covers the virtual building blocks — veth pairs, loopback, bonds, and dummy
interfaces — that every later topology (bridges, routers, overlays, containers) is assembled from.
