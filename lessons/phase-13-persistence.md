# Phase 13: Configuration Persistence

Everything you've built so far disappears on reboot — namespaces, addresses, routes, bridges,
firewall rules, sysctls. This phase covers how to make a network configuration **declarative and
persistent**, primarily with systemd-networkd, plus the other tools you'll meet in the field.
