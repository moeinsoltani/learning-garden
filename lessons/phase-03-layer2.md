# Phase 3: Layer-2 Networking

Now you wire the virtual interfaces together into switched networks. This phase builds the
Linux bridge (a software switch), adds VLAN segmentation, and explores MACVLAN and TAP — the
interfaces that give containers and virtual machines a presence on the wire.
