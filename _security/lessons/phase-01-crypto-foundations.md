# Phase 1: Cryptography Foundations

Before certificates, TLS, or OAuth make any sense, you need the primitives they're built from:
symmetric and asymmetric encryption, hashing and MACs, key exchange, digital signatures, and
where randomness comes from. This phase is hands-on with `openssl` and stays focused on *what each
primitive guarantees* and *what it does not*.
